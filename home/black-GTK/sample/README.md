# Black-Is-The-New-Black

Black GTK 2,3 theme. Forked from [Flat-Remix-GTK](https://github.com/daniruiz/Flat-Remix-GTK) theme. 

Almost complete black with a touch of gray.

**Installation**

- Clone this repo
- cd into the directory
- `sudo make install`
